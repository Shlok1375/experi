# Experi

Free A/B experiment design tool for startup PMs.

Live at: https://tryexperi.netlify.app

## What it does

- Sample size and runtime calculator
- CUPED variance reduction (saves 30-40% of users)
- Bayesian probability alongside frequentist p-values
- Sequential testing with O'Brien-Fleming boundaries
- SRM detection before you interpret results
- Live experiment analyzer — paste mid-run data, get a verdict
- Experiment risk score out of 100
- Stakeholder digest generator (Slack, Email, Notion)
- Segment explorer for heterogeneous treatment effects

## Built with

Vanilla HTML, CSS, JavaScript. No dependencies. Single file.

## Background

Built from patterns observed running experiments across startups
and research at Purdue University.
